### Week 6 Achievements:

- Mastering Advanced IAM Deep-Security Access Control:
  - Successfully configured Condition blocks based on static IP addresses (aws:SourceIp) and system timestamps (aws:CurrentTime), effectively revoking access for audit accounts immediately upon project expiration or when accessed outside the office.
  - Successfully applied Permissions Boundaries to place strict upper-limit constraints on sub-users, completely eliminating the risk of unauthorized privilege escalation even if the user is attached to wide Administrator policies.
  - Successfully deployed the EC2 Instance Profile solution, granting Python source code secure access to S3 via IAM Roles and completely eliminating the security risks of hardcoding Access Keys in the source code

- Landing Zone and Single Sign-On (SSO) Deployment:
  - Successfully activated AWS Organizations and configured the centralized AWS IAM Identity Center portal using group-based permissions.
  - Smoothly operated Permission Sets connecting enterprise accounts; mastered automated AWS CLI authentication via the aws configure sso command and safe session logout using aws sso logout.
  - Fully automated the querying, management, and mass creation of Users/Groups by programming interactions with Identity Store APIs using the Python boto3 library.

- Resource Categorization via Tags and Security Evaluation via Security Hub:
  - Successfully executed tagging, resource filtering, and automated grouping of virtual machines by tags (tag-based groups) into the MarketingBU Resource Group using both the Console and AWS CLI.
  - Successfully operated AWS Config in coordination with AWS Security Hub to run automated configuration scans aligned with international standards such as AWS Foundational Security Best Practices and PCI DSS.
  - Familiarized with the Auditor role, interpreting risk graphs (Findings), and practicing the remediation or disabling of misconfigurations to optimize security scores on the overview Dashboard.

- Realizing Cost-Optimized EC2 ChatOps with Lambda:
  - Successfully built a serverless automation system: Wrote Python boto3 code on AWS Lambda to accurately detect system tags and execute automated scale-down or scale-up cycles for EC2 virtual machines.
  - Successfully integrated Amazon EventBridge as a scheduled trigger to automatically start and stop servers, helping enterprises cut over 60% of wasted costs from Test/Staging environments.
  - Successfully established a Webhook connection pipeline to push visually color-coded server status notifications (Stopping/Starting) to Slack communication channels in real time
