# WMS Pro: AWS Cloud Infrastructure Deployment Guide

### Workshop Objectives
This workshop provides step-by-step instructions for provisioning standard **AWS Cloud** infrastructure to run the **WMS Pro** platform:
- Deploy an **Amazon RDS PostgreSQL** relational database for waybill & financial data.
- Provision an **Amazon EC2 (Ubuntu 24.04 LTS)** virtual server for the NestJS Backend and Next.js Web Frontend.
- Configure network firewall rules (**Security Groups**) allowing secure EC2-to-RDS communication.

### Prerequisites
1. An active **AWS Account** (preferably within the AWS Free Tier).
2. A modern web browser (Chrome/Edge/Firefox) to access the **AWS Management Console**.
3. Basic knowledge of Linux commands, SSH, and PostgreSQL database logic.
